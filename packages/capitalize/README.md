# @thinice/capitalize

Capitalizes the first letter of a string.

## Installation

```bash
npm install @thinice/capitalize
```

## Usage

```typescript
import { capitalize } from "@thinice/capitalize";

capitalize("hello"); // "Hello"
capitalize("world"); // "World"
capitalize(""); // ""
capitalize("a"); // "A"
capitalize("Hello"); // "Hello" (already capitalized)
```
