# @thinice/is-node

Distinguishes Node vs browser vs edge-ish runtime.

## Installation

```bash
npm install @thinice/is-node
```

## Usage

```typescript
import { isNode } from "@thinice/is-node";

isNode(); // true in Node.js, false in browser
```
