# @thinice/is-ci

Checks common CI environment variables.

## Installation

```bash
npm install @thinice/is-ci
```

## Usage

```typescript
import { isCi } from "@thinice/is-ci";

isCi(); // true if running in CI (GitHub Actions, GitLab CI, etc.)
```
