# @thinice/last-item

Returns the last item in an array (with bounds check).

## Installation

```bash
npm install @thinice/last-item
```

## Usage

```typescript
import { lastItem } from "@thinice/last-item";

lastItem([1, 2, 3]); // 3
lastItem(["a", "b"]); // "b"
lastItem([]); // undefined
```
